ใส่ "ตา" ให้หุ่นยนต์เห็นวัตถุข้างหน้า

📝 ลงชื่อเข้าเรียน

RP20402 ชั่วโมง

🎯 ด่านที่ 1: HC-SR04 ทำงานยังไง?

สวัสดี ! วันนี้เพิ่ม "เซ็นเซอร์ตา" ให้หุ่นยนต์ — มันจะรู้ว่ามีอะไรอยู่ข้างหน้าไกลแค่ไหน

🦇 หลักการ: คลื่นเสียงสะท้อน (เหมือนค้างคาว!)

📤

1. ส่ง (TRIG)

ขา TRIG ส่งคลื่นเสียง 40 kHz (สูงเกินกว่าหูคนได้ยิน)

🔄

2. สะท้อน

คลื่นกระทบวัตถุ → สะท้อนกลับ

📥

3. รับ (ECHO)

ขา ECHO รับคลื่นกลับ → คำนวณเวลาที่ใช้ → แปลงเป็นระยะทาง

🧠
คณิตศาสตร์ที่ใช้: เสียงเดินทาง 343 ม./วินาที ในอากาศ → ระยะทาง = (เวลา × 343) / 2 (หาร 2 เพราะคลื่นไป-กลับ)

📊 ข้อมูลเทคนิค

ระยะวัด2 cm ถึง 400 cm (4 เมตร)
ความแม่นยำ±0.3 cm
มุมตรวจจับ15° (รูปกรวย)
ไฟเลี้ยง+5V
จำนวนสาย4 เส้น (VCC, TRIG, ECHO, GND)

🔌 ด่านที่ 3: การต่อสาย + โค้ด CircuitPython

HC-SR04 ต่อกับ Maker Pi RP2040 ผ่าน GP Port (Grove cable หรือสายจัมเปอร์)

🐍 โค้ดอ่านระยะ

import board import adafruit_hcsr04 import time # สร้างเซ็นเซอร์ — TRIG=GP3, ECHO=GP2 sonar = adafruit_hcsr04.HCSR04( trigger_pin=board.GP3, echo_pin=board.GP2 ) while True: try: d = sonar.distance # คืนค่า cm print("ระยะ:", d, "cm") except RuntimeError: print("วัดไม่ได้") # บางครั้งคลื่นหายไป time.sleep(0.2)
⚠️
ทำไมต้อง try/except? ถ้าระยะ > 4 เมตร หรือคลื่นไม่กลับมา จะเกิด RuntimeError — ใช้ try/except กันโปรแกรมพัง

🐍 โค้ดประยุกต์: แสดงสถานะด้วย LED

import board, neopixel, adafruit_hcsr04, time sonar = adafruit_hcsr04.HCSR04(trigger_pin=board.GP3, echo_pin=board.GP2) pixels = neopixel.NeoPixel(board.GP18, 2) while True: try: d = sonar.distance if d < 20: pixels.fill((255, 0, 0)) # แดง = อันตราย elif d < 50: pixels.fill((255, 255, 0)) # เหลือง = ระวัง else: pixels.fill((0, 255, 0)) # เขียว = ปลอดภัย print(d, "cm") except RuntimeError: pass time.sleep(0.1)

🚨 ด่านที่ 4: ระบบเตือน (Sonar Beeper)

ทำให้บัซเซอร์ "ปี๊บถี่ขึ้นเรื่อยๆ" เมื่อวัตถุใกล้ — เหมือน "เซ็นเซอร์ถอยรถ"!

🐍 โค้ด Parking Sensor

import board, pwmio, adafruit_hcsr04, time sonar = adafruit_hcsr04.HCSR04(trigger_pin=board.GP3, echo_pin=board.GP2) buzzer = pwmio.PWMOut(board.GP22, frequency=880, duty_cycle=0) while True: try: d = sonar.distance if d < 100: # คำนวณช่วงเงียบ: ใกล้ → เงียบสั้น = ปี๊บถี่ silence = d / 100 # 0.0 ถึง 1.0 buzzer.duty_cycle = 32768 time.sleep(0.05) buzzer.duty_cycle = 0 time.sleep(silence) else: buzzer.duty_cycle = 0 # ไกล = เงียบ time.sleep(0.1) except RuntimeError: pass
🚗
ของจริง: โค้ดนี้คือหัวใจของระบบ "เซ็นเซอร์ถอยรถ" ในรถยนต์! รถยุคใหม่ทุกคันมีเซ็นเซอร์แบบนี้

📝 ด่านที่ 5: ใบงานที่ 12 — "เซ็นเซอร์ถอยรถ"

🎯 ภารกิจ: ระบบเซ็นเซอร์ถอยรถ (10 คะแนน)

สร้างระบบจริง — จับวัตถุข้างหน้าแล้วเตือนผู้ใช้

✅ ต้องทำ:
  1. วัดระยะด้วย HC-SR04 ตลอดเวลา
  2. ระยะ < 10 cm → LED แดง + บัซเซอร์ ปี๊บถี่ๆ + แสดง "DANGER!" ที่ Console
  3. 10-30 cm → LED เหลือง + บัซเซอร์ ปี๊บช้าๆ
  4. 30-100 cm → LED เขียวอ่อน + ไม่มีเสียง
  5. > 100 cm → LED ดับ + ไม่มีเสียง

🐍 โครงโค้ดให้เริ่ม

import board, neopixel, pwmio, adafruit_hcsr04, time sonar = adafruit_hcsr04.HCSR04(trigger_pin=board.GP3, echo_pin=board.GP2) pixels = neopixel.NeoPixel(board.GP18, 2) buzzer = pwmio.PWMOut(board.GP22, frequency=880, duty_cycle=0) def beep(times, gap): for _ in range(times): buzzer.duty_cycle = 32768; time.sleep(0.05) buzzer.duty_cycle = 0; time.sleep(gap) while True: try: d = sonar.distance print(d, "cm") if d < 10: # TODO: LED แดง + บัซเซอร์ปี๊บถี่ + print "DANGER!" pass elif d < 30: # TODO: LED เหลือง + ปี๊บช้า pass elif d < 100: # TODO: LED เขียวอ่อน + ไม่มีเสียง pass else: # TODO: LED ดับ + ไม่มีเสียง pass except RuntimeError: pass time.sleep(0.05)
📸 แคปหน้าจอ + วิดีโอส่งครู:
  1. แคปโค้ดเต็ม ที่เขียนเสร็จ
  2. แคปหน้า Simulator ขณะที่วัตถุอยู่ < 10 cm → LED แดง
  3. แคปหน้า Simulator ขณะที่วัตถุอยู่ 30-100 cm → LED เขียวอ่อน
  4. วิดีโอ ≥30 วินาที ของบอร์ดจริงทำงาน (ถ้ามี) แสดงทุกระดับ
  5. แคป Console ขณะที่แสดงค่าระยะ + คำว่า "DANGER!"
