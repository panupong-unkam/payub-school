🎯 ด่านที่ 1: IR Sensor ทำงานยังไง?

สวัสดี ! วันนี้เซ็นเซอร์อันใหม่ — IR (Infrared) ใช้ "แสงอินฟราเรด" ที่ตาคนมองไม่เห็น แต่กล้องมือถือเห็น!

🌈 หลักการ: แสงสะท้อนจากพื้น

⚪ พื้นสีขาว/อ่อน

สะท้อนแสง IR ดี → ตัวรับเห็นแสงเยอะ → ค่า = 0 (LOW)

⚫ พื้นสีดำ/เข้ม

ดูดซับแสง ดี → ตัวรับแทบไม่เห็นอะไร → ค่า = 1 (HIGH)

⚠️
สับสนได้! Sensor บางรุ่นกลับด้าน (ดำ=0, ขาว=1) ต้องดู datasheet หรือทดสอบจริง — โมดูล Cytron Maker Line Follower ของเรา: ดำ=1, ขาว=0

🔌 IR Sensor ที่ใช้: Maker Line

Cytron Maker Line คือเซ็นเซอร์ 5 ตัวรวมในแถบเดียว — เห็นตำแหน่งเส้นได้ละเอียด!

เซ็นเซอร์ใต้พื้นค่าที่ได้ความหมาย
S1 (ขวาสุด)ดำ1เห็นเส้นด้านขวา
S2 (กลาง)ขาว0ไม่เห็นเส้น
S3 (ซ้ายสุด)ขาว0ไม่เห็นเส้น
💡
โครงการของเรา ใช้ Maker Line แบบ 3 หัวจริง ๆ (ซ้าย/กลาง/ขวา) → ต่อกับ GP4, GP5, GP6 — เพียงพอสำหรับเรียน!

🐍 โค้ดอ่านค่า IR Sensor

import board, digitalio, time # IR sensors 3 ตัว — ซ้าย กลาง ขวา ir_left = digitalio.DigitalInOut(board.GP4) ir_left.direction = digitalio.Direction.INPUT ir_mid = digitalio.DigitalInOut(board.GP5) ir_mid.direction = digitalio.Direction.INPUT ir_right = digitalio.DigitalInOut(board.GP6) ir_right.direction = digitalio.Direction.INPUT while True: L = ir_left.value M = ir_mid.value R = ir_right.value print(L, M, R) time.sleep(0.1)

🛣️ ด่านที่ 3: ตรรกะหุ่นยนต์ตามเส้น

หลักการง่ายๆ — มี 3 ตา → ตาไหนเห็นเส้น เลี้ยวไปทางนั้น

LMRหมายความสั่งทำ
010เส้นอยู่กลาง⬆ เดินหน้า
100เส้นเอียงไปซ้าย⬅ เลี้ยวซ้าย
001เส้นเอียงไปขวา➡ เลี้ยวขวา
110เลี้ยวซ้ายชัด↩ เลี้ยวซ้ายแรง
011เลี้ยวขวาชัด↪ เลี้ยวขวาแรง
000หลุดเส้น!🛑 หยุด/หาเส้น

🐍 โค้ด Line Follower เต็ม

import board, digitalio, pwmio, time # --- Setup IR sensors --- sensors = [] for pin in [board.GP4, board.GP5, board.GP6]: s = digitalio.DigitalInOut(pin) s.direction = digitalio.Direction.INPUT sensors.append(s) # --- Setup motors (เหมือนเดิม) --- m1a = pwmio.PWMOut(board.GP8, frequency=10000) m1b = pwmio.PWMOut(board.GP9, frequency=10000) m2a = pwmio.PWMOut(board.GP10, frequency=10000) m2b = pwmio.PWMOut(board.GP11, frequency=10000) SPEED = 0.5 def set_motor(a, b, s): if s >= 0: a.duty_cycle = int(s * 65535); b.duty_cycle = 0 else: a.duty_cycle = 0; b.duty_cycle = int(abs(s) * 65535) def drive(left, right): set_motor(m2a, m2b, left) set_motor(m1a, m1b, right) while True: L = sensors[0].value M = sensors[1].value R = sensors[2].value print(L, M, R) if M and not L and not R: drive(SPEED, SPEED) # กลาง — เดินหน้า elif L and not R: drive(SPEED*0.3, SPEED) # เห็นซ้าย — เลี้ยวซ้าย elif R and not L: drive(SPEED, SPEED*0.3) # เห็นขวา — เลี้ยวขวา else: drive(0, 0) # หลุด — หยุด time.sleep(0.02)
⚡
เทคนิคสำคัญ: drive(SPEED*0.3, SPEED) = ล้อซ้ายช้า + ล้อขวาเร็ว → หุ่นยนต์ เลี้ยวซ้ายอย่างนุ่มนวล ไม่ใช่หมุนตัว

🤖
🔵 ซ้าย: - 🔵 ขวา: - 🎮 สั่งทำ: หยุด 📊 ระยะที่เดิน: 0 px
👁️
สังเกต: เซ็นเซอร์เป็นสี เขียว = อยู่บนพื้นขาว (ค่า=0), สี แดง = อยู่บนเส้นดำ (ค่า=1) — หุ่นยนต์ปรับทิศตามที่ตาแต่ละข้างเห็น

📝 ด่านที่ 5: ใบงานที่ 14 — "Line Follower Race"

🎯 ภารกิจ: หุ่นยนต์ตามเส้นแข่ง (15 คะแนน)

ทำให้หุ่นยนต์เดินตามเส้นโค้งได้สำเร็จ + บันทึกเวลา

✅ เกณฑ์ที่ต้องมี:
  1. ใช้ IR Sensor 3 ตัว ที่ GP4, GP5, GP6 (3 คะแนน)
  2. หุ่นยนต์เดินตามเส้น ไม่ออกนอกเส้น ตลอดเส้นทาง (5 คะแนน)
  3. เลี้ยวโค้งได้ นุ่มนวล ไม่กระชาก (3 คะแนน)
  4. มี NeoPixel แสดงทิศที่กำลังเลี้ยว (เขียว=ตรง, น้ำเงิน=ซ้าย, ส้ม=ขวา) (2 คะแนน)
  5. เมื่อ หลุดเส้น → หยุด + บัซเซอร์ดัง (2 คะแนน)

🐍 โครงโค้ดให้เริ่ม

import board, digitalio, pwmio, neopixel, time # --- Setup IR sensors, motors, LED, buzzer --- ir_L = digitalio.DigitalInOut(board.GP4); ir_L.direction = digitalio.Direction.INPUT ir_M = digitalio.DigitalInOut(board.GP5); ir_M.direction = digitalio.Direction.INPUT ir_R = digitalio.DigitalInOut(board.GP6); ir_R.direction = digitalio.Direction.INPUT pixels = neopixel.NeoPixel(board.GP18, 2) buzzer = pwmio.PWMOut(board.GP22, frequency=880, duty_cycle=0) # --- motors เหมือนเดิม --- SPEED = 0.5 def drive(left, right): # TODO: ใช้ set_motor pass while True: L = ir_L.value; M = ir_M.value; R = ir_R.value print(L, M, R) if M and not L and not R: pixels.fill((0, 255, 0)) # เขียว = ตรง drive(SPEED, SPEED) elif L: pixels.fill((0, 0, 255)) # น้ำเงิน = ซ้าย # TODO: drive(เลี้ยวซ้ายนุ่ม) elif R: pixels.fill((255, 100, 0)) # ส้ม = ขวา # TODO: drive(เลี้ยวขวานุ่ม) else: # หลุดเส้น — TODO: หยุด + บัซเซอร์ดัง pass time.sleep(0.02)
