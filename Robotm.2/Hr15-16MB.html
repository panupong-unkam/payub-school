เข็มทิศดิจิทัล (Compass)

เซ็นเซอร์แม่เหล็กในตัวบอร์ด

📝 กรุณาลงชื่อเข้าเรียน

micro:bit2 ชั่วโมง

🎯 ด่านที่ 1: เข็มทิศใน micro:bit คืออะไร?

สวัสดี ! สัปดาห์นี้เราจะใช้ เข็มทิศดิจิทัล หาทิศเหนือเหมือนนักสำรวจสมัยก่อน — แต่ใช้ชิปจิ๋วแทน!

🧲 Magnetometer (เซ็นเซอร์แม่เหล็ก)

ชิปที่อยู่หลังบอร์ด ทำหน้าที่วัด สนามแม่เหล็กของโลก เพื่อหาทิศเหนือ

  • วัด มุม 0-359 องศา เทียบกับทิศเหนือ
  • 0° = หันหน้าไปทางทิศเหนือ
  • 90° = ทิศตะวันออก, 180° = ทิศใต้, 270° = ทิศตะวันตก
⚠️ ต้อง Calibrate ก่อนใช้!

ครั้งแรกที่เรียกใช้เข็มทิศ บอร์ดจะขอให้คุณ "เอียงให้ครบทุกทิศ" เพื่อปรับค่าก่อน — ทำตามทีเดียวพอ ใช้ตลอดทั้ง session

🔧 ด่านที่ 2: Calibrate (ปรับเทียบ) เข็มทิศ

การ Calibrate คือการ "สอน" เข็มทิศให้รู้จักสนามแม่เหล็กของพื้นที่ที่เราอยู่

📋 ขั้นตอน Calibrate

  1. เปิดบอร์ดและรันโปรแกรมที่เรียก compass heading
  2. จอจะแสดงข้อความ "TILT TO FILL SCREEN"
  3. เอียงบอร์ด ให้จุด LED แต่ละดวงค่อย ๆ เปิดขึ้น (ทำเป็นวงกลม)
  4. เมื่อไฟครบทั้ง 25 ดวง → จอแสดง หน้ายิ้ม = พร้อมใช้!

📚 บล็อก calibrate compass

on start: calibrate compass // เรียกครั้งเดียวตอนเริ่ม
💡 เกร็ดความรู้:

ถ้าใกล้สิ่งของเหล็ก/แม่เหล็ก/มือถือ — เข็มทิศจะเพี้ยน! ทดสอบในที่โล่ง

🎯 ด่านที่ 4: ลูกศรนำทาง 4 ทิศ

เทคนิคที่ใช้บ่อยที่สุด — แปลงมุมเป็น ทิศหลัก 4 ทิศ (N E S W) แล้วแสดงลูกศร

📚 โค้ดเต็ม

on start: calibrate compass forever: set h to (compass heading) if (h < 45 or h > 315): show arrow North // ทิศเหนือ else if (h < 135): show arrow East // ทิศตะวันออก else if (h < 225): show arrow South // ทิศใต้ else: show arrow West // ทิศตะวันตก
🧠 วิเคราะห์ช่วงมุม:
  • เหนือ (N): 315° → 0° → 45° (เป็นช่วงข้าม 0)
  • ตะวันออก (E): 45° → 135°
  • ใต้ (S): 135° → 225°
  • ตะวันตก (W): 225° → 315°
