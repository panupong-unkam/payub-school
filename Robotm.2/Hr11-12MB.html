🎯 ด่านที่ 1: Accelerometer คืออะไร?

สวัสดี ! วันนี้เราจะใช้เซ็นเซอร์ที่ "รับรู้การเคลื่อนไหว" ของบอร์ดได้

📡 Accelerometer (เซ็นเซอร์ความเร่ง)

ชิปจิ๋วที่อยู่หลังบอร์ด ทำหน้าที่ วัดความเอียง + การสั่น + การเคลื่อนที่ ของบอร์ดในแกน X, Y, Z

💡 เซ็นเซอร์แบบเดียวกับในมือถือ ที่หมุนหน้าจอตามที่เอียง

📐 3 แกน

  • X — ซ้าย-ขวา (เอียงข้าง)
  • Y — หน้า-หลัง (ก้ม-เงย)
  • Z — บน-ล่าง (เขย่าขึ้น-ลง)

🎯 ใช้ทำอะไรได้?

  • เครื่องนับก้าว (Step counter)
  • เกมเขย่าทอยลูกเต๋า
  • ตรวจจับการตกของบอร์ด
  • รีโมตควบคุมด้วยการเอียง

📊 ด่านที่ 3: อ่านค่า X, Y, Z ตรง ๆ

ถ้าต้องการความละเอียดสูง ใช้บล็อก acceleration (mg) อ่านค่าตัวเลขโดยตรง

📏 หน่วย mg (มิลลิ-g)

  • 1000 mg = 1g = แรงโน้มถ่วงโลก
  • วางบอร์ดราบกับโต๊ะ → Z ≈ -1000 mg, X≈0, Y≈0
  • เอียงซ้าย 45° → X ≈ -700 mg

📚 ตัวอย่าง: แสดงค่า X ตลอดเวลา

forever: show number (acceleration x) pause 200 ms clear screen
🧠 ใช้กับเงื่อนไข:
if (acceleration x > 500): show arrow East // เอียงขวามากๆ else if (acceleration x < -500): show arrow West // เอียงซ้ายมากๆ else: clear screen
