💡 ด่านที่ 2: เลือกโปรเจกต์ของคุณ!

มี 6 ไอเดียให้เลือก หรือคิดเองก็ได้ — คลิกเลือกที่ชอบ

🚗
A: หุ่นยนต์ส่งของอัจฉริยะ
ใช้: Motor + HC-SR04 + Servo + LED + บัซเซอร์
วิ่งไปข้างหน้า เลี่ยงสิ่งกีดขวาง พอถึงปลายทางใช้ Servo "ปล่อยของ" + ส่งเสียง
⭐⭐⭐⭐
🏎️
B: หุ่นยนต์ Line Racing
ใช้: Motor + IR Sensor 3 ตัว + LED
เดินตามเส้นได้เร็วและแม่นยำ — แข่งวัดเวลากับเพื่อน
⭐⭐⭐
🚪
C: ประตูบ้านอัจฉริยะ
ใช้: HC-SR04 + Servo + LED + บัซเซอร์
คนเดินเข้าใกล้ < 30 cm → Servo เปิดประตู + LED เขียว + กล่าว "ยินดีต้อนรับ"
⭐⭐⭐
🤖
D: หุ่นยนต์เล่นเพลง + แสง
ใช้: ปุ่ม + บัซเซอร์ + NeoPixel + Servo
เปียโนมินิ — กดปุ่มได้ 6 โน้ต + LED เปลี่ยนสีตามโน้ต + Servo โบกตามจังหวะ
⭐⭐
🦮
E: ไม้เท้านำทาง (ผู้พิการ)
ใช้: HC-SR04 + บัซเซอร์ + NeoPixel
วัดระยะข้างหน้าตลอด → ใกล้สิ่งกีดขวาง เสียงเตือนถี่ขึ้น + LED แดง (เหมือนเซ็นเซอร์ถอยรถ)
⭐⭐
🎮
F: เกม Reaction Game
ใช้: NeoPixel + ปุ่ม + บัซเซอร์
LED สุ่มเปลี่ยนสี — ผู้เล่นต้องกดปุ่มให้ตรงสี → จับเวลา reaction time
⭐⭐
💭
มีไอเดียของตัวเอง? ปรึกษาครูได้! ขอให้: (1) ใช้อุปกรณ์ ≥3 ตัว (2) มีตรรกะ if/else ชัดเจน (3) ทำได้จริงใน 2 สัปดาห์

🗺️ ด่านที่ 3: วางแผนโปรเจกต์ (Planning Canvas)

ตอบ 5 คำถามนี้ในใจ (หรือเขียนลงสมุด) ก่อนเริ่มเขียนโค้ด

1. 🎯 ชื่อโปรเจกต์ + เป้าหมาย

ตัวอย่าง: "หุ่นยนต์ส่งของ" — วิ่งจาก A ไป B โดยไม่ชนอะไร

2. 🔧 อุปกรณ์ที่ใช้ + ขา GPIO

ตัวอย่าง: HC-SR04 (GP2, GP3), Motor 1 (GP8, GP9), Servo (GP12), LED (GP18)

3. 🧠 ตรรกะการทำงาน (เป็นภาษาไทย)

ตัวอย่าง: ถ้าระยะ < 20 → หยุด ถอย เลี้ยว / ถ้าระยะ ≥ 20 → เดินหน้า

4. 📦 สิ่งที่ต้องเตรียม (ฮาร์ดแวร์)

ตัวอย่าง: บอร์ด, สาย USB, ถ่าน, เทปกาว, กล่องกระดาษทำตัวรถ

5. ⚠️ ความท้าทายที่คาดว่าจะเจอ

ตัวอย่าง: หุ่นยนต์อาจติดมุม / มอเตอร์ไม่หมุน / เซ็นเซอร์เพี้ยน
