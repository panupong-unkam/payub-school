🎯 ด่านที่ 1: ทบทวน + เป้าหมายวันนี้

ยินดีต้อนรับกลับ ! สัปดาห์ที่แล้วเราใช้จอ LED แล้ว วันนี้เราจะ "รับ input" จากปุ่ม

🎯 เป้าหมายวันนี้

  1. ใช้ ปุ่ม A, B, A+B รับ input จากผู้ใช้
  2. สร้าง ตัวแปร (Variable) เก็บค่าตัวเลข
  3. ทำ ตัวนับ (Counter) ที่กดแล้ว +1
  4. เริ่มมีปฏิสัมพันธ์กับโลกภายนอกแล้ว!
💭 เปรียบเทียบ:

ในชีวิตจริง คีย์บอร์ดและเมาส์ = ปุ่ม A/B
ความจำในสมอง = ตัวแปร

🔘 ด่านที่ 2: ปุ่ม A, B, A+B

บนบอร์ดมีปุ่มซ้าย (A) และขวา (B) — ใช้บล็อก on button pressed ดักจังหวะการกด

กดปุ่ม!

🎮 ลองคลิกปุ่ม

📚 บล็อก 3 แบบ

on button A pressed: show string "A" on button B pressed: show string "B" on button A+B pressed: // กด 2 ปุ่มพร้อมกัน show icon Heart
💡 จดจำ: "on button ... pressed" เป็น Event block — ทำงานก็ต่อเมื่อมีคนกดปุ่ม ไม่ต้องใส่ใน forever

📦 ด่านที่ 3: ตัวแปร (Variable)

ตัวแปร = "กล่องเก็บค่า" ที่มีชื่อ — เปลี่ยนค่าได้ ใช้ใหม่ได้ตลอดโปรแกรม

🔹 set (ตั้งค่า)

set score to 0

→ สร้างกล่องชื่อ score เก็บเลข 0

🔹 change by (เพิ่ม)

change score by 1

→ เอาค่าเดิม + 1 (counter!)

🧠 ตัวอย่าง: ตัวแปร score

เริ่มต้น: score = 0 กด A: change score by 1 → score = 1 กด A: change score by 1 → score = 2 กด A: change score by 1 → score = 3 แสดง: show number score → จอแสดง "3"
📌 กฎตั้งชื่อตัวแปร: ใช้ภาษาอังกฤษ ไม่มีเว้นวรรค ขึ้นต้นด้วยตัวอักษร เช่น score, count, myScore

🔢 ด่านที่ 4: รวม ปุ่ม + ตัวแปร = ตัวนับ

รวมทุกอย่างเข้าด้วยกัน → ได้ Click Counter เครื่องนับคลิกที่ใช้ได้จริง!
