🎯 ด่านที่ 1: DC Motor ต่างจาก Servo ยังไง?

สวัสดี ! สัปดาห์นี้เราจะใช้ DC Motor — มอเตอร์อีกแบบที่ "หมุนตลอด" ไม่หยุดที่มุม

คุณสมบัติServo SG90DC Motor
การหมุนหมุนถึงมุมแล้วหยุด (0-180°)หมุนต่อเนื่อง ไม่หยุด
ความเร็วช้า (ใช้เวลาเปลี่ยนมุม)เร็วมาก (100+ รอบ/นาที)
การควบคุมกำหนดมุม (angle)กำหนดความเร็ว (-1 ถึง +1)
ทิศทางกำหนดผ่านมุม+ = เดินหน้า, − = ถอยหลัง
ใช้ทำแขนกล, กรงเล็บ, ก้านเข็มล้อรถ, พัดลม, ปั๊ม

🔌 DC Motor ของเราต่อกับอะไร?

Maker Pi RP2040 มี Motor Driver ในตัว (DRV8833) — ขับมอเตอร์ 2 ตัวพร้อมกัน ที่:

🅼 MOTOR 1

  • ขั้ว M1A = ขา GP8
  • ขั้ว M1B = ขา GP9
  • ใช้กับ ล้อขวา ของหุ่นยนต์

🅼 MOTOR 2

  • ขั้ว M2A = ขา GP10
  • ขั้ว M2B = ขา GP11
  • ใช้กับ ล้อซ้าย ของหุ่นยนต์
✅
Motor Driver ทำอะไร? ชิป DRV8833 ทำหน้าที่ "ขยายไฟ" จากสัญญาณ 3.3V ของ RP2040 ให้เป็น 5-6V ที่มอเตอร์ต้องการ + กลับขั้วไฟได้ ทำให้มอเตอร์หมุน 2 ทิศทาง

⚙️ ด่านที่ 2: หมุนยังไง? เดินหน้า/ถอย/หยุด

หลักการ: ส่งสัญญาณ PWM (0 ถึง 1) ให้ขา A หรือ B ของมอเตอร์

⚙️
เดินหน้า (Forward)

M1A = 1.0 (เต็มกำลัง)
M1B = 0

⚙️
ถอยหลัง (Reverse)

M1A = 0
M1B = 1.0 (เต็มกำลัง)

🛑
หยุด (Brake)

M1A = 0
M1B = 0

🐍 โค้ดควบคุม DC Motor ด้วย CircuitPython

import board import pwmio # Motor 1 = ขา GP8 (M1A) + GP9 (M1B) m1a = pwmio.PWMOut(board.GP8, frequency=10000) m1b = pwmio.PWMOut(board.GP9, frequency=10000) # เดินหน้าเต็มกำลัง — duty 65535 = 100% m1a.duty_cycle = 65535 m1b.duty_cycle = 0 # ถอยหลังเต็มกำลัง — สลับขั้ว m1a.duty_cycle = 0 m1b.duty_cycle = 65535 # หยุด — ทั้งคู่ = 0 m1a.duty_cycle = 0 m1b.duty_cycle = 0
🧠
ทำไม frequency=10000? Motor Driver ชอบความถี่สูงเพื่อให้มอเตอร์หมุนเรียบ — ต่างจาก Servo ที่ใช้ 50 Hz

🐍 ด่านที่ 4: เขียนฟังก์ชันสำหรับเคลื่อนที่

เขียนฟังก์ชัน forward(), backward(), left(), right(), stop() ครั้งเดียว ใช้ได้ตลอด

📚 โค้ดเต็ม (Copy ไปใช้ได้เลย)

import board, pwmio, time # Setup มอเตอร์ทั้ง 2 ตัว m1a = pwmio.PWMOut(board.GP8, frequency=10000) m1b = pwmio.PWMOut(board.GP9, frequency=10000) m2a = pwmio.PWMOut(board.GP10, frequency=10000) m2b = pwmio.PWMOut(board.GP11, frequency=10000) def set_motor(a, b, speed): """speed = -1.0 ถึง 1.0""" if speed >= 0: a.duty_cycle = int(speed * 65535) b.duty_cycle = 0 else: a.duty_cycle = 0 b.duty_cycle = int(abs(speed) * 65535) def forward(speed=0.7): set_motor(m1a, m1b, speed) # ขวา เดินหน้า set_motor(m2a, m2b, speed) # ซ้าย เดินหน้า def backward(speed=0.7): set_motor(m1a, m1b, -speed) set_motor(m2a, m2b, -speed) def turn_left(speed=0.7): set_motor(m1a, m1b, speed) # ขวาเดินหน้า set_motor(m2a, m2b, -speed) # ซ้ายถอย → หมุนซ้าย def turn_right(speed=0.7): set_motor(m1a, m1b, -speed) # ขวาถอย set_motor(m2a, m2b, speed) # ซ้ายเดินหน้า → หมุนขวา def stop(): set_motor(m1a, m1b, 0) set_motor(m2a, m2b, 0) # ทดลอง: เดินหน้า 2 วินาที แล้วหยุด forward(0.8) time.sleep(2) stop()
💡
เคล็ดลับการหมุน: ให้ล้อ 2 ฝั่ง หมุนคนละทิศ → หุ่นยนต์จะหมุนอยู่กับที่ (Tank Turn) ถ้าหยุดล้อข้างนึง + ขับอีกข้าง = หมุนช้าๆ (Pivot Turn)

📝 ด่านที่ 5: ใบงานที่ 11 — "หุ่นยนต์เต้นรำ"

🎯 ภารกิจ: หุ่นยนต์เต้นรำ (10 คะแนน)

เขียนโปรแกรมให้หุ่นยนต์ เต้นรำตามคิว 6 จังหวะติดต่อกัน:

  1. เดินหน้า 1 วินาที
  2. ถอยหลัง 1 วินาที
  3. หมุนซ้าย 0.5 วินาที
  4. หมุนขวา 0.5 วินาที
  5. หมุน 360° (หาเองว่าใช้เวลากี่วินาที!)
  6. หยุด + LED 2 ดวง กระพริบสีเขียว 3 ครั้ง

🐍 โครงโค้ดให้เริ่ม

import board, pwmio, neopixel, time # --- Setup มอเตอร์ + LED --- m1a = pwmio.PWMOut(board.GP8, frequency=10000) m1b = pwmio.PWMOut(board.GP9, frequency=10000) m2a = pwmio.PWMOut(board.GP10, frequency=10000) m2b = pwmio.PWMOut(board.GP11, frequency=10000) pixels = neopixel.NeoPixel(board.GP18, 2) def set_motor(a, b, speed): if speed >= 0: a.duty_cycle = int(speed * 65535); b.duty_cycle = 0 else: a.duty_cycle = 0; b.duty_cycle = int(abs(speed) * 65535) def forward(): set_motor(m1a, m1b, 0.7); set_motor(m2a, m2b, 0.7) def backward(): set_motor(m1a, m1b, -0.7); set_motor(m2a, m2b, -0.7) def left(): set_motor(m1a, m1b, 0.7); set_motor(m2a, m2b, -0.7) def right(): set_motor(m1a, m1b, -0.7); set_motor(m2a, m2b, 0.7) def stop(): set_motor(m1a, m1b, 0); set_motor(m2a, m2b, 0) # --- ลำดับการเต้น --- forward(); time.sleep(1) backward(); time.sleep(1) # TODO: เพิ่ม turn left/right/360°/หยุด/กระพริบ LED
📸 แคปหน้าจอ + วิดีโอส่งครู:

  2. วิดีโอ ≥30 วินาที หุ่นยนต์เต้นรำตามจังหวะ (ถ่ายจริง หรือใน Simulator)
