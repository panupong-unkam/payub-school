🤔 ด่านที่ 3: ทบทวน If / If-Else (เงื่อนไข)

เงื่อนไข = "ถ้า...แล้ว..." ใช้กับเซ็นเซอร์เพื่อให้หุ่นยนต์ตัดสินใจเอง

🔹 if (ถ้า)

ทำเมื่อเงื่อนไขเป็นจริงเท่านั้น

if (distance < 200) { stop driving }

→ ถ้าระยะ < 200 mm → หยุด

🔹 if-else (ถ้า...ไม่งั้น...)

มี 2 ทางเลือก ทำอย่างใดอย่างหนึ่ง

if (distance < 200) { turn right 90° } else { drive forward }

→ ใกล้กำแพง → เลี้ยว
→ ไกล → เดินหน้า

🧠 เทียบกับชีวิตจริง:

"ถ้าฝนตก แล้วกางร่ม ไม่งั้นใส่แว่นกันแดด" — เป็น if-else โดยมีคน "อ่านค่า" สภาพอากาศก่อนตัดสินใจ

🔗 ด่านที่ 4: รวม Loop + เงื่อนไข + เซ็นเซอร์

หุ่นยนต์ที่ "ฉลาดจริง" ต้องตรวจเซ็นเซอร์ ตลอดเวลา ไม่ใช่แค่ครั้งเดียว → ต้องใส่ใน Loop

🤖 ตัวอย่าง: หุ่นยนต์หลบกำแพงอัตโนมัติ

forever { if (Distance Sensor < 200 mm) { stop driving turn right 90 degrees } else { drive forward } }

วิเคราะห์:
→ forever = ตรวจตลอด
→ if = ถ้าใกล้กำแพง → หยุด + เลี้ยวขวา
→ else = ไม่งั้นเดินหน้าต่อ

⚠️ ระวัง 3 ข้อ:
  • ลืม forever → หุ่นยนต์ตรวจครั้งเดียวแล้วหยุด
  • ใส่ drive forward 200mm (มีระยะ) → หุ่นยนต์เดินจบก่อนเช็คเซ็นเซอร์
  • ตั้งระยะใกล้เกินไป (เช่น 50mm) → หุ่นยนต์ชนกำแพงก่อนเลี้ยว
