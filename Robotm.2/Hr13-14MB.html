🎯 ด่านที่ 1: เซ็นเซอร์ที่ไม่มีตัว แต่ใช้ได้!

สวัสดี ! วันนี้ เซ็นเซอร์น่าทึ่ง 2 ตัว ที่ใช้ฮาร์ดแวร์ "ในตัว" ไม่ใช่เซ็นเซอร์แยก

💡

เซ็นเซอร์แสง (Light)

ใช้ จอ LED 5×5 เป็นเซ็นเซอร์! เพราะ LED รับแสงได้ด้วย (เป็น LDR ขนาดเล็ก)

ค่า 0 = มืดสนิท | 255 = สว่างจ้า

🌡️

เซ็นเซอร์อุณหภูมิ (Temperature)

วัดอุณหภูมิ ใน CPU ของบอร์ด ไม่ใช่อุณหภูมิห้องโดยตรง

หน่วย °C (เซลเซียส)

⚠️ ข้อจำกัด:
  • เซ็นเซอร์อุณหภูมิ คลาดเคลื่อน ~2-3°C จากอุณหภูมิห้องจริง (เพราะวัดที่ CPU)
  • เซ็นเซอร์แสง วัดได้คร่าว ๆ ไม่แม่นยำเท่าเซ็นเซอร์แสงเฉพาะ

💡 ด่านที่ 2: อ่านค่าเซ็นเซอร์แสง

บล็อก light level คืนค่าระหว่าง 0 ถึง 255

🌑 มืด (0) 128 ☀️ สว่าง (255)

🎮 ลองเลื่อนดูค่า

📚 ตัวอย่างโค้ด

forever: show number (light level) pause 500 ms clear screen
🌙 ใช้กับเงื่อนไข — Night Light
forever: if (light level < 50): show icon Heart // มืด → เปิดไฟ else: clear screen // สว่าง → ปิดไฟ

🌡️ ด่านที่ 3: อ่านค่าเซ็นเซอร์อุณหภูมิ

บล็อก temperature (°C) คืนค่าอุณหภูมิเป็นองศาเซลเซียส

❄️ เย็น 28 °C 🔥 ร้อน

🎮 ลองเลื่อนดูค่า

📚 ตัวอย่างโค้ด

on button A pressed: show number (temperature) show string "C"
🚨 แจ้งเตือนอุณหภูมิ
forever: if (temperature > 30): show icon Sad // ร้อนเกิน play tone Middle C else if (temperature < 20): show icon Snow // เย็นเกิน else: show icon Happy // สบาย

🏠 ด่านที่ 4: รวมเซ็นเซอร์ — Smart Home Mini

ใช้ทั้ง 2 เซ็นเซอร์ สร้าง "ผู้ช่วยเฝ้าห้อง" — รายงานสภาพแสง + อุณหภูมิ ผ่านปุ่ม A/B

📚 โค้ดเต็ม

on button A pressed: // ตรวจแสง show string "L:" show number (light level) on button B pressed: // ตรวจอุณหภูมิ show string "T:" show number (temperature) show string "C" forever: // เตือนเมื่อร้อน if (temperature > 32): show icon Sad pause 1000 clear screen
💡 ฝึกคิด: ทำไมต้องใส่ pause + clear screen หลังแสดงไอคอนเศร้า?

→ ถ้าไม่ใส่ ไอคอนจะกระพริบรัวเพราะ forever วนเร็วมาก จนตาเรามองไม่ทัน
