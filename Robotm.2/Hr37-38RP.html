🐛 ด่านที่ 2: แก้บั๊กที่พบบ่อย (Top 5)

🐛 1. โค้ดรันไม่ขึ้น — ImportError

ขึ้นข้อความ ImportError: no module named 'adafruit_motor'

วิธีแก้: คุณยังไม่ติดตั้งไลบรารี → คัดลอกโฟลเดอร์ adafruit_motor จาก CIRCUITPY/lib หรือดาวน์โหลดจาก Adafruit Bundle
🐛 2. มอเตอร์ไม่หมุน

รันแล้วเงียบสนิท ไม่มีอะไรขยับ

วิธีแก้:
1. ตรวจสวิตช์ Power ที่ขั้วถ่าน (อยู่ใกล้ Motor Terminal)
2. ตรวจ duty_cycle ว่าใส่เลข > 0 หรือเปล่า
3. ลองเสียบมอเตอร์ที่ขั้ว M1A/M1B ไม่ใช่ M2
🐛 3. HC-SR04 อ่านค่าเป็น 0 หรือ Error ตลอด

เซ็นเซอร์เพี้ยน วัดอะไรไม่ได้

วิธีแก้:
1. ตรวจขา TRIG/ECHO ตรงกับโค้ดไหม
2. ตรวจ VCC = 5V ไม่ใช่ 3.3V
3. ใส่ try/except ครอบเสมอ
🐛 4. หุ่นยนต์หมุนไปทิศตรงข้าม

สั่งให้เลี้ยวซ้าย แต่หมุนขวา หรือเดินหน้าแต่ถอย

วิธีแก้: สลับสายมอเตอร์ที่ขั้ว M1A กับ M1B หรือสลับค่า speed (+ ↔ −) ในโค้ด
🐛 5. Servo สั่น ไม่หยุดที่มุม

Servo สั่นตลอดเวลาแม้สั่งแค่ครั้งเดียว

วิธีแก้:
1. ตรวจ frequency=50 ในการสร้าง PWM
2. ตรวจไฟเลี้ยง — Servo ต้องการ กระแสมาก ใช้ถ่าน 4 ก้อนแทน USB
🛠️
เทคนิคทอง: ใช้ print(...) ใส่ทุกที่ในโค้ด เพื่อดูว่า "ตอนนี้โปรแกรมไปถึงไหน + ตัวแปรค่าอะไร" — Debug ที่ดีที่สุดของโลก!
