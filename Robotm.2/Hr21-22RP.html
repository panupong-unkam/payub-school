💎 ด่านที่ 2: RP2040 ชิปคืออะไร?

RP2040 คือชิปประมวลผล (CPU) ที่ออกแบบโดย Raspberry Pi Foundation ประเทศอังกฤษ

  • เปิดตัวปี 2021 — ใหม่กว่า micro:bit
  • มี CPU 2 cores วิ่งเร็ว 133 MHz
  • RAM 264 KB · Flash 2 MB
  • GPIO 26 ขา — ใช้ต่อเซ็นเซอร์/มอเตอร์ได้เยอะ
  • รองรับ Python (CircuitPython/MicroPython) และ C++
RP2040 ARM Cortex-M0+
Dual Core
133 MHz
264 KB RAM

🤖 ด่านที่ 3: Cytron Maker Pi RP2040 บอร์ดของเรา

Cytron (บริษัทมาเลเซีย) เอาชิป RP2040 มาใส่บอร์ดสำหรับ "สร้างหุ่นยนต์" โดยเฉพาะ — มีทุกอย่างที่ต้องใช้ในบอร์ดเดียว

🧩 อุปกรณ์ที่มีในตัวบอร์ด (Built-in)

💡

NeoPixel RGB ×2

LED 16 ล้านสี ควบคุมแยกได้ทีละดวง — ใช้แสดงสถานะหุ่นยนต์

🔘

ปุ่ม GP0 + GP1

2 ปุ่มกดบนบอร์ด ใช้รับ input จากผู้ใช้

🔊

บัซเซอร์ Piezo

เล่นเสียง/เพลงได้ทันที ไม่ต้องต่อลำโพง

⚙️

Motor Driver ×2

ขับ DC Motor ได้ 2 ตัวพร้อมกัน — ทำหุ่นยนต์เคลื่อนที่

🎚️

Servo Ports ×4

เสียบ Servo Motor 4 ตัวพร้อมกัน (ขา GP12-15)

🔌

Grove Ports ×7

ต่อเซ็นเซอร์ภายนอก (HC-SR04, IR, ฯลฯ) ผ่านสาย Grove

✅
ดีกว่า micro:bit ตรงไหน? ทุกอย่างพร้อมในบอร์ดเดียว — ไม่ต้องบอร์ดขยาย ไม่ต้องเชื่อมสายเยอะ เริ่มทำหุ่นยนต์ได้เลย!

🐍 ตัวอย่างโค้ด CircuitPython แรก

# โค้ดแรก: ทำให้ NeoPixel เปลี่ยนสีแดง import board import neopixel # NeoPixel อยู่ที่ขา GP18, มี 2 ดวง pixels = neopixel.NeoPixel(board.GP18, 2) pixels[0] = (255, 0, 0) # ดวงที่ 0 = แดง pixels[1] = (0, 255, 0) # ดวงที่ 1 = เขียว
