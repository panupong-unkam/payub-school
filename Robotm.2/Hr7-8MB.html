🎯 ด่านที่ 1: micro:bit คืออะไร?

ยินดีต้อนรับสู่หน่วย 2 ! วันนี้เราจะเริ่ม จับบอร์ดคอมพิวเตอร์จิ๋วจริง เป็นครั้งแรก

💡 micro:bit คือ

บอร์ดคอมพิวเตอร์จิ๋วขนาดเท่าฝ่ามือ ออกแบบโดย BBC ประเทศอังกฤษ ให้นักเรียนใช้เรียนเขียนโปรแกรม

  • ราคาถูก ใช้ในโรงเรียนทั่วโลก
  • มีเซ็นเซอร์ในตัวพร้อมใช้ทันที
  • ไม่ต้องต่อสายอะไรเพิ่ม

📦 ในกล่องมีอะไร?

  • บอร์ด micro:bit v2
  • สาย USB (Micro-USB)
  • ที่ใส่ถ่าน AAA × 2 (ใช้เมื่อไม่ต่อ USB)

⚠️ หน่วยนี้ใช้แค่บอร์ดล้วน ไม่มีอุปกรณ์เสริม

🤔 เทียบกับ VEXcode VR:

VEXcode VR = หุ่นยนต์ เสมือน บนคอม | micro:bit = ฮาร์ดแวร์จริง ที่จับได้ ใส่ในกระเป๋าได้

🔍 ด่านที่ 2: ส่วนประกอบของ micro:bit v2

มาทำความรู้จักกับสิ่งที่อยู่บนบอร์ดทั้งหมด — ทุกอย่างที่เห็น ใช้ได้ทันที!

ด้านหน้า (Front)

  • 💡 จอ LED 5×5 (25 ดวง สีแดง)
  • 🔘 ปุ่ม A / ปุ่ม B
  • 🟡 โลโก้ทอง (Touch logo — v2)
  • 🎤 ไมโครโฟน (v2)

ด้านหลัง (Back)

  • 📡 Accelerometer (เซ็นเซอร์การเคลื่อนไหว)
  • 🧭 Compass (เข็มทิศ)
  • 🌡️ เซ็นเซอร์อุณหภูมิ (ใน CPU)
  • 📻 Radio + Bluetooth
  • 🔊 ลำโพง (v2)
  • 🔌 พอร์ต USB + GPIO 25 ขา
⚠️ ระวัง: จับบอร์ดที่ ขอบ เท่านั้น! ห้ามจับขาเหล็กหรือชิปเพราะไฟฟ้าสถิตจากนิ้วอาจทำชิปเสียหาย

🖥️ ด่านที่ 3: เปิด MakeCode Editor

เราจะเขียนโปรแกรมให้ micro:bit ผ่านเว็บไซต์ MakeCode (ไม่ต้องลงโปรแกรมที่เครื่อง)

🌐 เปิดที่:

https://makecode.microbit.org

🧩 ส่วนของหน้าจอ MakeCode

  • Simulator — จำลอง micro:bit ด้านซ้าย
  • Toolbox — หมวดบล็อกตรงกลาง
  • Workspace — พื้นที่วางบล็อกด้านขวา
  • Download — ดาวน์โหลดไฟล์ .hex ลงบอร์ด

📥 ขั้นตอนอัปโหลดลงบอร์ด

  1. เสียบบอร์ดกับคอมด้วยสาย USB
  2. กด Download ใน MakeCode
  3. ลากไฟล์ .hex ไปวางบน drive MICROBIT
  4. ไฟส้มกระพริบ → เสร็จ! บอร์ดรันโปรแกรมทันที

💡 ด่านที่ 4: เล่นกับจอ LED 5×5

จอ LED มี 25 ดวง — ใช้แสดง ตัวอักษร, ตัวเลข, ไอคอน, อนิเมชัน

หัวใจ ❤️ จากบล็อก show icon Heart

📚 บล็อกหลักของจอ LED

show string "Hello" // แสดงข้อความเลื่อน show number 42 // แสดงตัวเลข show icon ♥ // แสดงไอคอนสำเร็จรูป (15 แบบ) show leds // แสดง pattern ที่เราออกแบบเอง clear screen // ลบจอ plot x 2 y 2 // เปิดไฟ 1 ดวงที่ตำแหน่งกลาง unplot x 2 y 2 // ปิดไฟ 1 ดวง
📌 ตำแหน่ง LED: นับจาก (0,0) มุมบนซ้าย ถึง (4,4) มุมล่างขวา — แกน x (แนวนอน), y (แนวตั้ง)

📝 ด่านที่ 5: ใบงานที่ 3 — "ป้ายชื่ออัจฉริยะ"

🎯 โจทย์ภารกิจ

เปิด makecode.microbit.org → สร้างโปรเจกต์ใหม่ → เขียนโปรแกรม "ป้ายชื่ออัจฉริยะ" ที่:

✅ ต้องทำให้ได้:
  1. แสดง ชื่อเล่นของตัวเอง เลื่อนผ่านจอ (show string)
  2. หยุดสั้น ๆ → แสดง ไอคอน 1 ตัว ที่ชอบ (show icon)
  3. หยุดสั้น ๆ → แสดง เลขประจำตัว (show number)
  4. ใส่บล็อก forever ครอบ — วนตลอดเวลา
📤 วิธีส่งงาน:
  1. กด Share ใน MakeCode → คัดลอกลิงก์
  2. ถ่ายวิดีโอ หน้าจอ Simulator หรือ บอร์ดจริง ทำงาน (≥ 15 วินาที)
  3. เข้าเว็บ โรงครูเจมส์ → ระบบส่งงาน → ใบงาน "ป้ายชื่ออัจฉริยะ"
🌟 โบนัส (+2 คะแนน): ใช้บล็อก show leds วาด pattern ตัวอักษรย่อของชื่อตัวเอง แทนไอคอน
