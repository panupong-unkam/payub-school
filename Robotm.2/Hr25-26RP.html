🎯 ด่านที่ 1: Servo Motor คืออะไร?

สวัสดี ! วันนี้เราเรียน "มอเตอร์ตัวแรก" ที่ใช้ในหุ่นยนต์ — Servo Motor SG90

⚙️ Servo คืออะไร?

มอเตอร์ที่หมุนไป "มุมที่กำหนด" แล้วหยุด ไม่ใช่หมุนตลอด

  • หมุนได้ 0° - 180°
  • แม่นยำ ±1°
  • มีกลไกป้องกันการเลื่อน
  • ราคาถูก (~80 บาท)

🤖 ใช้ทำอะไรได้?

  • แขนกลของหุ่นยนต์
  • กรงเล็บจับของ (Gripper)
  • เลี้ยวล้อหน้าของรถ
  • ก้านเข็มของนาฬิกา
  • เปิด-ปิดประตูบ้านอัจฉริยะ

📚 โค้ดที่ใช้ตอนนี้

import board from adafruit_motor import servo import pwmio # Servo Port 1 ของ Maker Pi อยู่ที่ขา GP12 pwm = pwmio.PWMOut(board.GP12, frequency=50) my_servo = servo.Servo(pwm) my_servo.angle = 90 # หมุนไปที่ 90 องศา

🔌 ด่านที่ 3: Servo Ports บน Maker Pi

Maker Pi RP2040 มี Servo Port 4 ตัว ที่เสียบตรงได้เลย

📍 ตำแหน่งและขาของ Servo

Servo Port ขา GPIO การใช้งาน
SERVO 1GP12นิยมใช้กับ Servo ตัวหลัก
SERVO 2GP13Servo ตัวรอง
SERVO 3GP14Servo ตัวที่ 3
SERVO 4GP15Servo ตัวที่ 4

🧩 วิธีเสียบ Servo

  1. หา Servo Port ที่บอร์ด (มีลายพิมพ์ "SERVO 1" - "SERVO 4")
  2. ดูสีของสาย Servo: แดง=VCC, น้ำตาล=GND, ส้ม=Signal
  3. เสียบให้ตรงสี (แดงตรงกับ +, น้ำตาลตรงกับ −)
  4. เปิดสวิตช์ Servo Power (อยู่ข้างขั้วถ่าน)
⚠️
เสียบผิดสีจะเสียทันที! Servo SG90 ทนแรงดันได้แค่ +5V ถ้าต่อ + กับ − สลับกัน — มอเตอร์อาจไหม้

🐍 ด่านที่ 4: โค้ดขั้นสูง — Sweep + ปุ่มควบคุม

มาเขียนโค้ดที่ทำให้ Servo ขยับซ้ำๆ (Sweep) และให้ ปุ่มควบคุม Servo

📚 โค้ด 1: Sweep 0° → 180° → 0°

import board, pwmio, time from adafruit_motor import servo pwm = pwmio.PWMOut(board.GP12, frequency=50) my_servo = servo.Servo(pwm) while True: # หมุนซ้าย-ไป-ขวา for a in range(0, 181, 5): my_servo.angle = a time.sleep(0.02) # กลับขวา-ไป-ซ้าย for a in range(180, -1, -5): my_servo.angle = a time.sleep(0.02)

📚 โค้ด 2: ใช้ปุ่ม A/B เพิ่ม-ลดมุม

import board, pwmio, digitalio, time from adafruit_motor import servo pwm = pwmio.PWMOut(board.GP12, frequency=50) my_servo = servo.Servo(pwm) angle = 90 my_servo.angle = angle btn_a = digitalio.DigitalInOut(board.GP20) btn_a.pull = digitalio.Pull.UP btn_b = digitalio.DigitalInOut(board.GP21) btn_b.pull = digitalio.Pull.UP while True: if not btn_a.value and angle < 180: angle += 5 my_servo.angle = angle print("มุม:", angle) if not btn_b.value and angle > 0: angle -= 5 my_servo.angle = angle print("มุม:", angle) time.sleep(0.1)
🧠
ทำไม frequency=50? Servo SG90 ต้องรับสัญญาณ PWM ความถี่ 50 Hz เป็นมาตรฐาน (ทุก 20ms ส่งพัลส์ 1 ครั้ง) — ห้ามเปลี่ยน!

📝 ด่านที่ 5: ใบงานที่ 10 — "ที่กั้นไม้ไผ่"

🎯 ภารกิจ: ที่กั้นไม้ไผ่อัตโนมัติ (10 คะแนน)

สมมติว่าคุณกำลังสร้าง ที่กั้นทางเข้าลานจอดรถ โดยใช้ Servo เป็นแกนยกที่กั้นขึ้น-ลง

📋 ข้อกำหนด

  • เริ่มต้น: ที่กั้น ปิด (Servo = 0°)
  • กด ปุ่ม A → ที่กั้นเปิด (90°) + LED 2 ดวง สีเขียว + บัซเซอร์ดัง 1 ครั้ง
  • กด ปุ่ม B → ที่กั้นปิด (0°) + LED 2 ดวง สีแดง + บัซเซอร์ดัง 1 ครั้ง
  • เริ่มต้นทุกครั้ง LED = แดง

🏗️ Hardware ที่ใช้

  • Maker Pi RP2040 × 1
  • Servo SG90 ที่ Servo 1 (GP12)
  • NeoPixel ในตัว (GP18)
  • บัซเซอร์ในตัว (GP22)
  • ปุ่ม A (GP20), B (GP21)

🐍 โครงโค้ดให้เริ่ม

import board, pwmio, digitalio, neopixel, time from adafruit_motor import servo # --- Setup --- pwm = pwmio.PWMOut(board.GP12, frequency=50) gate = servo.Servo(pwm) pixels = neopixel.NeoPixel(board.GP18, 2) buzzer = pwmio.PWMOut(board.GP22, frequency=880, duty_cycle=0) btn_a = digitalio.DigitalInOut(board.GP20); btn_a.pull = digitalio.Pull.UP btn_b = digitalio.DigitalInOut(board.GP21); btn_b.pull = digitalio.Pull.UP # --- เริ่มต้น: ที่กั้นปิด + LED แดง --- gate.angle = 0 pixels[0] = (255, 0, 0) pixels[1] = (255, 0, 0) def beep(): buzzer.duty_cycle = 32768 time.sleep(0.15) buzzer.duty_cycle = 0 while True: if not btn_a.value: # TODO: เปิดที่กั้น 90° + LED เขียว + beep pass if not btn_b.value: # TODO: ปิดที่กั้น 0° + LED แดง + beep pass time.sleep(0.1)
📸 แคปหน้าจอ 5 รูปส่งครู:
  1. โค้ดเต็มที่เขียนเสร็จ (แคปทั้งหน้า)
  2. เริ่มต้น: Servo 0° + LED แดง (แคปบอร์ด)
  3. กดปุ่ม A: Servo 90° + LED เขียว (แคปบอร์ด)
  4. กดปุ่ม B: Servo 0° + LED แดง (แคปบอร์ด)
  5. ถ้าทำกับบอร์ดจริง: ภาพ Servo จริงเปิด-ปิด (ถ่ายภาพ)
