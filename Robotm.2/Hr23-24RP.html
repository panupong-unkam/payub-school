🌈 ด่านที่ 1: NeoPixel คืออะไร?

สวัสดี ! วันนี้เราเริ่ม "คุยกับฮาร์ดแวร์" จริงๆ แล้ว — มาเริ่มจากของสนุกที่สุดก่อน: ไฟ RGB!

💡 NeoPixel คือ

LED ที่ผสมแม่สี 3 สี (R G B) ในดวงเดียว → ผสมได้ 16.7 ล้านสี

  • Red (แดง) 0-255
  • Green (เขียว) 0-255
  • Blue (น้ำเงิน) 0-255

🎨 ผสมสี (Color Mix)

  • (255, 0, 0) = แดง
  • (0, 255, 0) = เขียว
  • (0, 0, 255) = น้ำเงิน
  • (255, 255, 0) = เหลือง (R+G)
  • (255, 0, 255) = ม่วง (R+B)
  • (255, 255, 255) = ขาว (R+G+B)
  • (0, 0, 0) = ดับ (ปิด)
🧠
ทำไม 255? เพราะคอมเก็บค่าเป็น byte (8 bit) = 0 ถึง 255. แต่ละสีปรับได้ 256 ระดับ → 256³ = 16.7 ล้านสี!

🔘 ด่านที่ 3: ปุ่ม GP0 + GP1 บนบอร์ด

Maker Pi RP2040 มีปุ่ม 2 ปุ่มที่ ขา GP20 และ GP21 (ใช่! ขาคนละเลขกับชื่อ — Cytron designed ไว้แบบนี้)

🎮 ลองคลิกปุ่มจำลอง:

📚 โค้ดอ่านค่าปุ่ม

import board import digitalio # ตั้งขา GP20 และ GP21 เป็น Input + ดึงค่า UP (ปกติ HIGH) btn_a = digitalio.DigitalInOut(board.GP20) btn_a.direction = digitalio.Direction.INPUT btn_a.pull = digitalio.Pull.UP btn_b = digitalio.DigitalInOut(board.GP21) btn_b.direction = digitalio.Direction.INPUT btn_b.pull = digitalio.Pull.UP while True: if not btn_a.value: # False = กดอยู่ (Active Low) print("กดปุ่ม A!") if not btn_b.value: print("กดปุ่ม B!")
⚠️
Active Low (สำคัญมาก!): ปุ่ม Maker Pi เมื่อ กด จะให้ค่า False (LOW) เมื่อ ไม่กด ให้ค่า True (HIGH) — ต้องใช้ if not btn.value

🔊 ด่านที่ 4: บัซเซอร์ Piezo (เล่นเพลง!)

📚 โค้ดเล่นโน้ตเดียว

import board import pwmio import time buzzer = pwmio.PWMOut(board.GP22, frequency=440, duty_cycle=0) # เปิดเสียง (duty 50%) buzzer.duty_cycle = 32768 time.sleep(0.5) # ปิดเสียง (duty 0) buzzer.duty_cycle = 0

📚 โค้ดเล่นเพลง Twinkle Twinkle

import board, pwmio, time buzzer = pwmio.PWMOut(board.GP22, frequency=440, duty_cycle=0) notes = [262, 262, 392, 392, 440, 440, 392] # C C G G A A G for n in notes: buzzer.frequency = n buzzer.duty_cycle = 32768 time.sleep(0.4) buzzer.duty_cycle = 0 time.sleep(0.1)
🎼
duty_cycle: ค่า 0 = ปิด, ค่า 32768 = 50% (เปิด-ปิดสลับเร็วครึ่งหนึ่งของเวลา) ใช้คุมความดัง — ค่า 65535 = เปิด 100% (ดังที่สุด)

📝 ด่านที่ 5: ใบงานที่ 9 — "RGB Show + บัซเซอร์"

🎯 ภารกิจ (10 คะแนน)

เขียนโปรแกรมใน Simulator (หรือบอร์ดจริง) ให้:

  1. เมื่อกด ปุ่ม A (GP20) → NeoPixel 2 ดวง เปลี่ยนสี ตามที่คุณเลือก + บัซเซอร์ เล่นโน้ต C (262 Hz)
  2. เมื่อกด ปุ่ม B (GP21) → NeoPixel 2 ดวง ดับ + บัซเซอร์ เล่นโน้ต G (392 Hz)
  3. เมื่อ ปล่อยมือ → เสียงหยุดทันที

🐍 โครงโค้ดให้เริ่ม

import board, neopixel, digitalio, pwmio, time pixels = neopixel.NeoPixel(board.GP18, 2) buzzer = pwmio.PWMOut(board.GP22, frequency=440, duty_cycle=0) btn_a = digitalio.DigitalInOut(board.GP20) btn_a.direction = digitalio.Direction.INPUT btn_a.pull = digitalio.Pull.UP # TODO: สร้าง btn_b แบบเดียวกัน while True: if not btn_a.value: # TODO: ตั้ง pixels[0] กับ pixels[1] เป็นสีที่ชอบ # TODO: ตั้ง buzzer.frequency = 262 และเปิดเสียง pass elif not btn_b.value: # TODO: ปิด pixels + เล่นโน้ต G pass else: buzzer.duty_cycle = 0 # หยุดเสียง
📸 แคปหน้าจอ 4 รูปส่งครู:
  1. โค้ดเต็มที่เขียนเสร็จ (แคปให้เห็นทั้งหน้า)
  2. กดปุ่ม A → LED ติดสีของคุณ (แคปบอร์ด)
  3. กดปุ่ม B → LED ดับ (แคปบอร์ด)
  4. ปล่อยมือ → LED กับเสียงตามสถานะ (แคปบอร์ด)
