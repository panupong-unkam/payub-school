5. สาระสำคัญ

หุ่นยนต์อัตโนมัติทำงานตามหลัก "Sense → Think → Act" = รับรู้ (เซ็นเซอร์) → คิด (เงื่อนไข) → ทำ (มอเตอร์/เซอร์โว) เป็นพื้นฐานของหุ่นยนต์ Roomba, รถยนต์ขับเอง Tesla, drone Algorithm หลบสิ่งกีดขวางอย่างง่าย: forever loop → วัดระยะ → ถ้าใกล้กำแพง (< 20 cm) → หยุด + ถอย + เลี้ยว → ถ้าไกล → เดินหน้า ค่า Threshold ที่ปรับได้: SAFE_DIST 20-30 cm, SPEED 0.4-0.7, BACK_TIME 0.3-0.8 วินาที, TURN_TIME 0.3-0.6 วินาที — ปรับให้เหมาะกับขนาดหุ่นยนต์และสนาม

  2. DC Motor + ฟังก์ชัน forward/backward/turn จากแผน 14
  3. การออกแบบ Flowchart
