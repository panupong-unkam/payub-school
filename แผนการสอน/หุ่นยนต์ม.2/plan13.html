2. จุดประสงค์การเรียนรู้
  1. อธิบายหลักการทำงานของ Servo Motor และความแตกต่างจาก DC Motor ได้ K
  2. เชื่อมต่อ Servo SG90 กับ Servo Port บนบอร์ด Maker Pi RP2040 ได้ถูกต้อง P
  3. ใช้ไลบรารี adafruit_motor.servo ควบคุม Servo ที่มุม 0-180° ได้ P
  4. เขียนโปรแกรม "ที่กั้นไม้ไผ่อัตโนมัติ" ที่ใช้ Servo + ปุ่ม + LED + บัซเซอร์ P
  5. ระมัดระวังเรื่องขั้วไฟ (VCC/GND) เพื่อความปลอดภัย A

5. สาระสำคัญ

Servo Motor SG90 คือมอเตอร์ที่หมุนไปที่ "มุมที่กำหนด" 0-180° แล้วหยุด มีกลไกป้องกันการเลื่อน ใช้ในแขนกล กรงเล็บ ที่กั้นรถ การต่อมีสาย 3 เส้น: แดง (VCC +5V), น้ำตาล/ดำ (GND), ส้ม/เหลือง (Signal) Maker Pi RP2040 มี Servo Port 4 ตัวที่ขา GP12-GP15 เสียบตรงได้ ใช้ pwmio.PWMOut(board.GP12, frequency=50) + servo.Servo(pwm) + my_servo.angle = N ตั้งมุม การส่งสัญญาณต้องเป็น 50 Hz (ทุก 20ms ส่ง 1 พัลส์)

▶ ชั่วโมงที่ 25 (60 นาที) : Servo Motor SG90

ขั้นที่ 1: การวิเคราะห์และกำหนดรายละเอียดของปัญหา (15 นาที)

  1. ครูเปิดเว็บไซต์ Robotm.2/Hr25-26RP.html
  2. ครูแสดงตัวอย่าง Servo จริงและสาธิตการหมุน

    3. เมื่อรถยนต์เลี้ยวซ้าย-ขวา ล้อหน้าเปลี่ยนทิศได้อย่างไร? ใช้มอเตอร์แบบไหน?

    น่าจะใช้ Servo เพราะต้องการให้หมุน "ถึงมุมแล้วหยุด" ไม่ใช่หมุนต่อเนื่องเหมือนล้อ

ขั้นที่ 2: การวางแผนการแก้ปัญหา (20 นาที)

  1. ครูแสดงสายของ Servo 3 เส้น และตำแหน่งของ Servo Port บน Maker Pi
  2. ครูสาธิตการเสียบ Servo ที่ Port 1 (GP12) — ต้องดูสีแดง-น้ำตาล-ส้ม
  3. ครูแนะนำโค้ด CircuitPython สำหรับ Servo:
    import board, pwmio from adafruit_motor import servo pwm = pwmio.PWMOut(board.GP12, frequency=50) my_servo = servo.Servo(pwm) my_servo.angle = 90

ขั้นที่ 3: การดำเนินการแก้ปัญหา (25 นาที)

  1. นักเรียนทดลอง Servo Demo ในเว็บไซต์ (slider 0-180°)
  2. นักเรียนเขียนโค้ด Sweep: for a in range(0, 181, 5): my_servo.angle = a
  3. นักเรียนทดสอบบนบอร์ดจริง
▶ ชั่วโมงที่ 26 (60 นาที) : ภารกิจ "ที่กั้นไม้ไผ่"

ขั้นที่ 3 (ต่อ): การดำเนินการแก้ปัญหา (40 นาที)

  1. ครูแสดงโจทย์ "ที่กั้นไม้ไผ่อัตโนมัติ":

    • กด A → Servo 90° + LED เขียว + beep
    • กด B → Servo 0° + LED แดง + beep

เฉลย (สำหรับครู)

import board, pwmio, digitalio, neopixel, time from adafruit_motor import servo pwm = pwmio.PWMOut(board.GP12, frequency=50) gate = servo.Servo(pwm) pixels = neopixel.NeoPixel(board.GP18, 2) buzzer = pwmio.PWMOut(board.GP22, frequency=880, duty_cycle=0) btn_a = digitalio.DigitalInOut(board.GP20); btn_a.pull = digitalio.Pull.UP btn_b = digitalio.DigitalInOut(board.GP21); btn_b.pull = digitalio.Pull.UP gate.angle = 0; pixels.fill((255,0,0)) def beep(): buzzer.duty_cycle = 32768; time.sleep(0.15); buzzer.duty_cycle = 0 while True: if not btn_a.value: gate.angle = 90; pixels.fill((0,255,0)); beep() if not btn_b.value: gate.angle = 0; pixels.fill((255,0,0)); beep() time.sleep(0.1)

ตอนที่ 4: for a in range(0, 91, 5): gate.angle = a; time.sleep(0.05)
