2. จุดประสงค์การเรียนรู้
  1. อธิบายหลักการผสมสีแบบ RGB (Red, Green, Blue) ค่า 0-255 ได้ K
  2. เขียนโค้ด CircuitPython ควบคุม NeoPixel เปลี่ยนสีและกระพริบได้ P
  3. ใช้ digitalio.DigitalInOut อ่านค่าปุ่ม (Active Low) ได้ P
  4. ใช้ pwmio.PWMOut ควบคุมบัซเซอร์ Piezo เล่นเสียงตามความถี่ได้ P
  5. เห็นความเชื่อมโยงระหว่างเทคโนโลยีและศิลปะ (สี + เสียง) A

5. สาระสำคัญ

NeoPixel เป็น LED ที่ผสมแม่สี 3 สี (R, G, B) ในดวงเดียว แต่ละสีปรับได้ 0-255 ค่า ผสมได้ 16.7 ล้านสี (256³) บอร์ด Maker Pi RP2040 มี NeoPixel 2 ดวงที่ขา GP18 ส่วนปุ่ม A และ B อยู่ที่ GP20 และ GP21 ทำงานแบบ Active Low (กด=False, ไม่กด=True) ใช้ digitalio.Pull.UP ดึงค่าขึ้น บัซเซอร์ Piezo อยู่ที่ GP22 ใช้ pwmio.PWMOut ส่งสัญญาณความถี่ (Hz) — โน้ตดนตรีคือความถี่เฉพาะ (C=262, D=294, E=330, F=349, G=392, A=440, B=494)

  3. ความรู้พื้นฐานเรื่องโน้ตดนตรี (โด เร มี)

ขั้นที่ 3 (ต่อ): การดำเนินการแก้ปัญหา (40 นาที)

  1. ครูอธิบายปุ่ม Active Low: btn = digitalio.DigitalInOut(board.GP20) + btn.pull = digitalio.Pull.UP + if not btn.value:
  2. ครูสาธิตเปียโน 7 คีย์ในเว็บไซต์ (ลองกดดูเสียง C-B)

import board, neopixel, digitalio, pwmio, time pixels = neopixel.NeoPixel(board.GP18, 2) buzzer = pwmio.PWMOut(board.GP22, frequency=440, duty_cycle=0) btn_a = digitalio.DigitalInOut(board.GP20); btn_a.pull = digitalio.Pull.UP btn_b = digitalio.DigitalInOut(board.GP21); btn_b.pull = digitalio.Pull.UP while True: if not btn_a.value: pixels.fill((255, 100, 0)) buzzer.frequency = 262 buzzer.duty_cycle = 32768 elif not btn_b.value: pixels.fill((0, 0, 0)) buzzer.frequency = 392 buzzer.duty_cycle = 32768 else: buzzer.duty_cycle = 0
