2. จุดประสงค์การเรียนรู้
  1. อธิบายหลักการสื่อสารไร้สายผ่านคลื่นวิทยุ 2.4 GHz ได้ K
  2. ใช้บล็อก radio set group, radio send, on radio received ได้ P
  3. ออกแบบและเขียนโปรเจกต์รวมความรู้หน่วย 2 (เป่ายิ้งฉุบ/Magic 8-Ball/Step Counter/Chat) P
  4. ทำงานเป็นทีม 2 คน (1 บอร์ดต่อคน) สื่อสารระหว่างกันได้ P, A
  5. เห็นความสำคัญของการตั้งกลุ่ม Radio เพื่อความเป็นส่วนตัวและความปลอดภัย A

  1. ความสามารถในการคิด
  2. ความสามารถในการแก้ปัญหา
  3. ความสามารถในการสื่อสาร (ทำงานเป็นทีม)
  4. ความสามารถในการใช้เทคโนโลยี
5. สาระสำคัญ

Radio ใน micro:bit ใช้คลื่นความถี่ 2.4 GHz (เดียวกับ WiFi, Bluetooth, ไมโครเวฟ) ส่ง-รับข้อมูลระหว่างบอร์ดในระยะ ~70 เมตร โดยตั้ง radio set group 0-255 — บอร์ดที่อยู่กลุ่มเดียวกันจะคุยกันได้ ส่งได้ทั้งตัวเลข ข้อความ และคู่ key-value โดยใช้ radio send + on radio received เป็นเทคโนโลยีพื้นฐานของ Internet of Things (IoT) ที่ใช้ในระบบสมาร์ทโฮม วิทยุสื่อสาร และ Walkie-Talkie

6. ความรู้เดิมที่นักเรียนต้องมี
  1. ความรู้ทั้งหมดจากแผนที่ 4-8 (จอ LED, ปุ่ม, ตัวแปร, Accelerometer, แสง/อุณหภูมิ, เข็มทิศ)
  2. การทำงานเป็นทีม
7. การจัดกิจกรรมการเรียนรู้
▶ ชั่วโมงที่ 17 (60 นาที) : Radio พื้นฐาน

ขั้นที่ 1: การวิเคราะห์และกำหนดรายละเอียดของปัญหา (15 นาที)

  1. ครูเปิดเว็บไซต์ Robotm.2/Hr17-18MB.html
  2. ครูตั้งคำถาม

    3. ก่อนมือถือมีอินเทอร์เน็ต เจ้าหน้าที่ตำรวจ/ทหารใช้อะไรในการสื่อสารระยะไกล? ทำงานอย่างไร?

    วิทยุสื่อสาร (Walkie-Talkie) ครับ/ค่ะ ใช้คลื่นวิทยุส่ง-รับเสียง — ต่างจากมือถือที่ใช้สัญญาณเซลล์ผ่านเสาส่ง

ขั้นที่ 2: การวางแผนการแก้ปัญหา (20 นาที)

  1. ครูแนะนำบล็อก Radio 3 ตัวหลัก:
    • radio set group N — ตั้งกลุ่ม
    • radio send number/string — ส่ง
    • on radio received — รับ
  2. ครูให้นักเรียนจับคู่ 2 คน (1 บอร์ด/คน) และตกลงเลขกลุ่ม (เช่น คู่ที่ 1 = กลุ่ม 1, คู่ที่ 2 = กลุ่ม 2)

ขั้นที่ 3: การดำเนินการแก้ปัญหา (25 นาที)

  1. นักเรียนเขียน Chat พื้นฐาน (กด A ส่ง "HI", กด B ส่ง "BYE", รับแล้วแสดงบนจอ)
  2. นักเรียนทดสอบโดยให้ทั้ง 2 บอร์ดมีโค้ดเดียวกัน
▶ ชั่วโมงที่ 18 (60 นาที) : มินิโปรเจกต์ปิดหน่วย micro:bit

ขั้นที่ 3 (ต่อ): การดำเนินการแก้ปัญหา (40 นาที)

  1. นักเรียนเลือกโปรเจกต์ 1 ใน 4:
    • A: เป่ายิ้งฉุบไร้สาย (Radio + Accelerometer + LED)
    • B: Magic 8-Ball (Accelerometer + LED)
    • C: Step Counter (Accelerometer + ตัวแปร + ปุ่ม)
    • D: Radio Chat 3 ทาง (Radio + ปุ่ม + LED)
  2. นักเรียนทำงานเป็นทีมและเขียนโปรแกรม

ขั้นที่ 4: การตรวจสอบและประเมินผล (20 นาที)

  1. นักเรียนทำใบงานที่ 9.1 (Final micro:bit Project)
  2. นักเรียนทำแบบทดสอบสรุปหน่วย 5 ข้อ
  3. นักเรียนแต่ละคู่/ทีมนำเสนอโปรเจกต์ + ครูประเมินตาม Rubric 10 คะแนน

    4. ถ้าเราตั้งกลุ่ม Radio = 0 ทั้งห้อง จะเกิดอะไร? เหตุใดต้องตั้งกลุ่มเฉพาะคู่?

    ทุกบอร์ดจะคุยกันมั่ว ข้อความปะปนกัน ตั้งกลุ่มเฉพาะคู่เพื่อให้ปลอดภัยและเป็นส่วนตัว เหมือนตั้งช่องวิทยุ

  4. ครูสรุปบทเรียนและประกาศว่าสัปดาห์หน้าสอบกลางภาค
