5. สาระสำคัญ

HC-SR04 เป็นเซ็นเซอร์อัลตราโซนิกใช้คลื่นเสียงความถี่ 40 kHz (สูงเกินกว่าหูคน) วัดระยะแบบเดียวกับค้างคาว/โลมา หลักการ: ขา TRIG ส่งคลื่นไป → กระทบวัตถุ → สะท้อนกลับ → ขา ECHO รับเวลา → คำนวณระยะ (เสียง 343 m/s, หาร 2 เพราะไป-กลับ) ใช้ได้ 2 cm - 400 cm มุมตรวจจับ 15° การต่อต้องใช้ VCC = 5V (ไม่ใช่ 3.3V) ใช้ไลบรารี adafruit_hcsr04 + ครอบด้วย try/except ป้องกัน RuntimeError

6. ความรู้เดิมที่นักเรียนต้องมี

  2. การใช้เงื่อนไข if-elif-else
  3. ความเร็วของเสียงในอากาศ (วิทยาศาสตร์ ม.1)

▶ ชั่วโมงที่ 29 (60 นาที) : HC-SR04 และคลื่นอัลตราโซนิก

ขั้นที่ 1: การวิเคราะห์และกำหนดรายละเอียดของปัญหา (15 นาที)

  1. ครูเปิดเว็บไซต์ Robotm.2/Hr29-30RP.html

    2. ค้างคาวบินในที่มืดสนิทได้อย่างไรไม่ชนของ? ใช้เทคโนโลยีอะไร?

    ใช้ระบบ Echolocation — ส่งคลื่นเสียงและฟังเสียงสะท้อนกลับ คำนวณระยะของวัตถุ ทำให้บินได้ในที่มืด

เฉลย (สำหรับครู)

import board, neopixel, pwmio, adafruit_hcsr04, time sonar = adafruit_hcsr04.HCSR04(trigger_pin=board.GP3, echo_pin=board.GP2) pixels = neopixel.NeoPixel(board.GP18, 2) buzzer = pwmio.PWMOut(board.GP22, frequency=880, duty_cycle=0) while True: try: d = sonar.distance print(d, "cm") if d < 10: pixels.fill((255,0,0)); buzzer.duty_cycle = 32768 time.sleep(0.05); buzzer.duty_cycle = 0; time.sleep(0.1) print("DANGER!") elif d < 30: pixels.fill((255,255,0)); buzzer.duty_cycle = 32768 time.sleep(0.05); buzzer.duty_cycle = 0; time.sleep(0.5) elif d < 100: pixels.fill((0,100,0)) buzzer.duty_cycle = 0 else: pixels.fill((0,0,0)); buzzer.duty_cycle = 0 except RuntimeError: pass time.sleep(0.05)
