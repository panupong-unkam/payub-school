  4. ซ้อมพูด 5-7 นาทีด้วย Timer ในเว็บไซต์ P

5. สาระสำคัญ

การสร้างโปรเจกต์ใหญ่ต้องผ่านขั้นตอน Debug หรือ "แก้บั๊ก" — Top 5 บั๊กที่พบ: (1) ImportError, (2) Motor ไม่หมุน (ลืมเปิด Power), (3) HC-SR04 เพี้ยน (VCC ผิด), (4) มอเตอร์ทิศกลับ, (5) Servo สั่น เทคนิคทอง: ใช้ print() ตรวจค่าตัวแปรทุกที่ การนำเสนอใช้รูปแบบ 6 ขั้น: (1) Hook ดึงดูดสนใจ → (2) ปัญหา + วิธีแก้ → (3) อุปกรณ์ + การทำงาน → (4) Demo สด (สำคัญที่สุด!) → (5) บทเรียนที่ได้ → (6) Q&A สำคัญที่สุดคือ "ถ้า Demo ไม่ขึ้น ให้เปิดวิดีโอสำรอง"

ขั้นที่ 2: เรียนรู้ Top 5 Bug Fixes (15 นาที)

  1. ครูสอน Debug Tips 5 หัวข้อในเว็บไซต์:
    • ImportError → คัดลอกโฟลเดอร์ adafruit_motor
    • มอเตอร์ไม่หมุน → ตรวจสวิตช์ Power
    • HC-SR04 Error → VCC = 5V
    • มอเตอร์ทิศกลับ → สลับสาย M1A กับ M1B
    • Servo สั่น → frequency = 50 + ใช้ถ่าน
  2. ครูเน้น: ใช้ print() เป็น Debug ที่ดีที่สุด

ขั้นที่ 3: ดำเนินการเขียนโค้ด + Debug (35 นาที)

  1. นักเรียนเขียนโค้ด MAIN LOOP ต่อจาก SETUP ในแผน 18
  2. ครูเดินดูเป็นรายบุคคล ช่วย Debug บั๊กที่เจอ
  3. นักเรียนบันทึกบั๊กที่พบและวิธีแก้ลงในใบงาน
