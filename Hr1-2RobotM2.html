ด่านที่ 1: "พิมพ์เขียว" ที่หุ่นยนต์เข้าใจ

ยินดีต้อนรับคุณ !
หุ่นยนต์ทำงานตาม อัลกอริทึม (Algorithm) หรือลำดับความคิดที่เราวางแผนไว้ให้ เพื่อความแม่นยำ วิศวกรจึงต้องวาดแผนผังความคิดที่เรียกว่า Flowchart ก่อนลงมือเขียนโค้ดเสมอ

ดูแอนิเมชันนี้เพื่อเข้าใจว่าโปรแกรมประมวลผลการทำงานเป็นลำดับอย่างไรครับ

ด่านที่ 2: ถอดรหัส 8 สัญลักษณ์สากล

มาทำความรู้จักกับสัญลักษณ์มาตรฐานวิศวกรรม ที่รูปทรงมีความหมายชัดเจน ห้ามใช้สลับกันเด็ดขาด!

Terminator
จุดเริ่มต้น (Start) และสิ้นสุด (End) ของโปรแกรม (วาดเป็นวงรีหรือแคปซูลเท่านั้น)
Process
การกระทำ หรือประมวลผล เช่น "มอเตอร์หมุน", "บวกเลข"
Decision
การตัดสินใจแบบมีเงื่อนไข ต้องมีทางแยก 2 ทางเสมอคือ ใช่ (Yes) / ไม่ (No)
Input / Output
รับค่าจากเซ็นเซอร์ (Input) หรือแสดงผลข้อมูล (Output)
Manual Input
รับข้อมูลจากการกดปุ่มสวิตช์ หรือพิมพ์คีย์บอร์ดของมนุษย์
Display
แสดงผลข้อความออกทางหน้าจอคอมพิวเตอร์ หรือ จอ LCD
Flowline
ลูกศรเชื่อมต่อแสดงทิศทางการทำงาน ต้องมีหัวลูกศรชี้ทางชัดเจน
Connector
จุดเชื่อมต่อลูกศรหลายๆ เส้นมารวมกันก่อนส่งไปสัญลักษณ์ถัดไป

ด่านที่ 3: โครงสร้างตรรกะ 3 รูปแบบหลัก

แผนผังด้านล่างถูกวาดขึ้นอย่างถูกต้องตามหลักวิศวกรรม 100% สังเกตทิศทางของลูกศร การแยกทาง และจุดรวมเส้นให้ดีนะครับ

1. ลำดับ (Sequence)

ทำงานตรงไปตรงมา จากบนลงล่าง

2. เลือกทำ (If-Else)

มีทางแยกให้ตัดสินใจ (ใช่ หรือ ไม่ใช่)

3. ทำซ้ำ (Loop)

วนกลับไปทำซ้ำ จนกว่าเงื่อนไขจะบอกให้หยุด

ด่านที่ 4: เจาะลึก "ถังขยะอัจฉริยะ"

มาดูการทำงานของโครงงานยอดฮิตกันครับ อัลกอริทึมด้านซ้ายแสดงการใช้ "Loop" ครอบ "If-Else" เพื่อให้เซ็นเซอร์คอยตรวจจับมือคนตลอดเวลาโดยไม่มีวันหลับ! (แผนผังนี้ถูกต้อง 100%)

อัลกอริทึม ถังขยะอัจฉริยะ

💡 เกร็ดความรู้:
เมื่อดูวิดีโอจบแล้ว ให้ลองเทียบจังหวะที่มือเข้าใกล้ถัง กับจังหวะในลูกศร Flowchart (เส้น YES) ดูนะครับ จะเห็นว่าการทำงานสอดคล้องกันเป๊ะ!

ด่านที่ 5: ภารกิจ "ไฟถนนอัตโนมัติ"

🛠️ ภารกิจวิศวกรฝึกหัด

จงวาด Flowchart ควบคุมไฟถนนด้วย LDR โดยมีเงื่อนไขดังนี้:

1. [เริ่มต้น] ระบบเริ่มทำงาน (ใช้สัญลักษณ์อะไร?)
2. [รับข้อมูล] อ่านค่าแสงจาก เซ็นเซอร์แสง LDR
3. [เงื่อนไข] สภาพแวดล้อม "มืด" ใช่หรือไม่?
   ➔ ถ้า ใช่ (มืด) : ให้ เปิดหลอดไฟ
   ➔ ถ้า ไม่ใช่ (สว่าง) : ให้ ปิดหลอดไฟ
4. [ทำซ้ำ] ลากเส้นวนลูปกลับไปทำข้อ 2 ตลอดเวลา
📌 คำแนะนำ: วาดโครงสร้างคล้ายถังขยะในด่านที่ 4 เลยครับ (แต่มีคำสั่งประมวลผลทั้งฝั่ง Yes และ No) วาดใส่เว็บไซต์https://app.diagrams.net/ แล้วส่งงานครูภาณุพงศ์ อุ่นคำใน ระบบส่งงานที่ขึ้นในระบบของเว็บไซต์

ดูคลิปตัวอย่างการทำงาน เพื่อเป็นไอเดียในการวาดครับ
